<task type="auto">
  <name>Task 2: Create OCR processor</name>
  <files>
    src/utils/ocrProcessor.ts
    src/types/ocr.ts
  </files>
  <action>
    Build ML Kit OCR wrapper with field extraction:
    
    1. Create src/types/ocr.ts:
       - OCRResult interface:
         * blocks: OCRBlock[]
         * fullText: string
         * confidence: number
       - OCRBlock interface:
         * text: string
         * confidence: number
         * boundingBox: {x, y, width, height}
       - ExtractedFields interface:
         * merchant: {value, confidence}
         * date: {value, confidence}
         * amount: {value, confidence}
         * vat: {value, confidence}
         * currency: {value, confidence}
       
    2. Create src/utils/ocrProcessor.ts:
       - processReceiptImage(imagePath): Promise<ExtractedFields>
         * Call ML Kit OCR on image
         * Get text blocks with confidence scores
         * Extract merchant:
           - Heuristic: Largest text at top of receipt
           - Look for business name patterns
         * Extract date:
           - Regex patterns: DD/MM/YYYY, DD-MM-YYYY
           - Look for "DATA" label
           - Validate: not in future, within 1 year
         * Extract amount:
           - Regex: "TOTALE", "IMPORTO", "EUR"
           - Look for largest number
           - Parse with Italian number format (1.234,56)
         * Extract VAT:
           - Look for "IVA" + number
           - Optional field
         * Calculate overall confidence
         * Return structured data
       
       - validateExtractedData(data): ValidationResult
         * Check merchant not empty
         * Check amount > 0
         * Check date valid
         * Return confidence score
       
       - preprocessImage(imagePath): Promise<string> (optional)
         * Resize to reduce processing time
         * Enhance contrast
         * Return processed image path
  </action>
  <verify>
    1. TypeScript compiles
    2. Functions handle test images
    3. Extraction heuristics documented
  </verify>
  <done>
    - ocrProcessor.ts with ML Kit integration

  </done>
